Chaincode Cards store public or encrypted data like emails, names, secrets, or credentials.

  • Use the Create tab to make a new card
  • Use the Wallet to view, export, or delete cards
  • Export bundles as encrypted `.zip.enc` files
  • Import cards or keys later using Import or Keychain

Security Tips: Always store keys securely. Use strong passphrases. All data stays on your device.
